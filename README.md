# 💬 WOW Meal N Deal — Customer Support Chatbot

AI-powered customer support chatbot built specifically for WOW Meal N Deal restaurant.

## ✨ Features
- Real-time chat interface like WhatsApp
- Specific to WOW Meal N Deal menu and info
- Conversation history — remembers context
- Typing animation with 5 second delay (human-like)
- Responds in both Urdu and English
- Clear chat option

## 🍔 Business Info Covered
- Full menu items
- Location and timings
- Delivery and pickup options
- Pricing information
- Contact details

## 🛠️ Tech Stack
- Python + Flask
- LangChain + Groq API (llama-3.1-8b-instant)
- HTML + CSS + JavaScript

## 🚀 How to Run
1. pip install -r requirements.txt
2. Add GROQ_API_KEY in .env file
3. python app.py
4. Open: http://127.0.0.1:5001
